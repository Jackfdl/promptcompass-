# WhichAI brand assets

These files are local so the interface remains fast, private and usable offline. CSS in
`styles.css` normalizes icon height and optical weight without stretching the artwork.

- Most SVG icons and wordmarks come from [Lobe Icons 1.94.0](https://github.com/lobehub/lobe-icons), MIT licensed.
- Sarvam icon and wordmark come from the official [Sarvam brand kit](https://www.sarvam.ai/brand).
- Reka AI and Thinking Machines Lab use their official site favicons, preserved inside local SVG wrappers.
- Product names, logos and wordmarks are trademarks of their respective owners. They identify the compared tools and do not imply endorsement.

When an official product wordmark is available, WhichAI uses it. Otherwise the UI keeps the
model name as live text and pairs it with the official provider icon or wordmark as the visual fallback.
